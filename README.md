# 约会亚洲女孩终极指南 - 网站落地页

一个设计精美、现代化的单页面网站，提供关于跨文化约会的建议和技巧。

## 📋 项目特色

- **时尚潮流设计**：采用粉色/紫色渐变主题，年轻活力的视觉风格
- **完全响应式**：完美适配桌面、平板和移动设备
- **丰富动画效果**：平滑滚动、渐入动画、悬停效果等
- **交互性强**：多种用户交互功能，提升用户体验
- **纯原生开发**：使用HTML5、CSS3和原生JavaScript，无任何框架依赖

## 🎨 技术栈

- HTML5
- CSS3（使用CSS变量、Grid、Flexbox、渐变、动画）
- 原生JavaScript（ES6+）
- Font Awesome 6.4.0（图标库）

## 📁 项目结构

```
web1/
├── index.html          # 主页面
├── css/
│   └── style.css      # 样式文件
├── js/
│   └── main.js        # JavaScript交互脚本
└── README.md          # 项目说明
```

## 🚀 快速开始

### 方法1：直接打开
双击 `index.html` 文件即可在浏览器中查看

### 方法2：使用本地服务器
```bash
# 如果已安装Python 3
python3 -m http.server 8000

# 或使用Python 2
python -m SimpleHTTPServer 8000

# 然后在浏览器访问 http://localhost:8000
```

### 方法3：使用VS Code Live Server
1. 安装 Live Server 扩展
2. 右键点击 index.html
3. 选择 "Open with Live Server"

## ✨ 交互功能

- **移动端菜单**：汉堡菜单切换
- **平滑滚动**：点击导航平滑滚动到对应区块
- **返回顶部**：滚动超过300px显示返回顶部按钮
- **滚动动画**：元素进入视口时触发渐入动画
- **鼠标跟随**：Hero区域渐变背景跟随鼠标移动
- **悬停效果**：卡片、按钮的悬停动画
- **波纹效果**：按钮点击波纹反馈
- **星级动画**：成功案例的星星逐个出现
- **图标旋转**：特色卡片悬停时图标360度旋转
- **表单提交**：邮件订阅功能（前端验证）

}
